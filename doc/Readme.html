distdiff

A tool for analyzing changes in Linux distributions

Distro Changes Analyzer (distdiff) is a tool for analyzing changes in Linux distributions. The tool compares old and new packages of the distribution and creates visual HTML report.

The tool is intended for Linux maintainers who are interested in ensuring compatibility of old and new versions of the distibution.

Supported Platforms

GNU/Linux, FreeBSD, Mac OS X

Dependencies

Requires:
  • Perl 5 (5.8 or newer)
  • PkgDiff (1.5 or newer)
  • GNU Diff
  • GNU Wdiff
  • GNU Awk
  • GNU Binutils (readelf)
  • RPM (rpm2cpio, rpm) for analysis of RPM packages
  • DPKG (dpkg-deb, dpkg) for analysis of DEB packages

Installation

The tool is ready-to-use after extracting the archive.

You can also use a Makefile to install the tool into the system:

 sudo perl Makefile.pl --install --prefix=PREFIX [/usr, /usr/local] 

This command will install a distdiff program to the PREFIX/bin system directory and private modules into the PREFIX/share.

Usage

Compare interface files (headers, libs, scripts, etc.) in packages:

 distdiff DIR1 DIR2 

DIR1 and DIR2 - directories with old and new packages of the Linux distribution (RPM, DEB, TAR.GZ, etc)

The HTML report will be generated to:

 distdiff_reports/DIR1_to_DIR2/changes_report.html 

Adv. Usage

Compare all files in packages:

 distdiff DIR1 DIR2 -all-files 

Print report to a file:

 distdiff DIR1 DIR2 -report-path FILE
